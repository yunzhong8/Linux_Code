# 分隔线
一行中有>=3个的星号/减号，不能有其他东西，符号可以有空格
```
***
* * * 
---
- --
```

# 列表
## 无序列表
用*\+\-符号作为标记改行为一行列表
```
*\+\-+space+第一行
*\+\-+space+第二行
```
 * 第一行
 * 第二行

## 有序列表
```
1.+space+第一项
2.+space+第二项
```
1. 第一项
2. 第二项
